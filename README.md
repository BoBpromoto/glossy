# Glossy Event Log Forensics
KDFS 2017 - 윈도우 이벤트 로그(.evtx) 분석 도구


## 기능
### KDFS 2017 요구사항
- 로그인/로그오프 이벤트
- 시스템 온/오프 이벤트
- 응용 프로그램 실행 정보
- 윈도우 서비스 정보
- 외부장치 연결 정보
- 시스템 시간 변경 정보

### 기타 추가 개발기능
- RDP 연결이력
- 이벤트 로그 삭제
- 윈도우 방화벽 로그
- 어플리케이션 에러
- 인쇄 기록
- 윈도우 업데이트 정보
- 사용자 계정 이벤트
- 소프트웨어 설치
- CD/DVD 쓰기 기록
- 문서 인쇄
- 무선 네트워크 연결정보
- 자동실행(스케줄러 / Run / BHO)
